* The endpoints for a minimal server with url handlers
** Start server
   #+BEGIN_SRC sh 
     cd ~/local/ghes-mirva-server
     go build && ./ghes-mirva-server --verbosity 4 start

     curl http://localhost:8080
     : Hi(base) 
   #+END_SRC
